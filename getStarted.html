How to get Started

Requirements

To use Billy you must have the software above installed on your computer

  • Java 6
  • Postgres (at least the 9.1 version)
  • Git
  • Maven 2

Download Billy

In order to get Billy you must be registered in Git, to access Billy repositories.
There are two ways to get Billy:

  1. Download the latest stable release, as a zip file from Billy's GitHub, and unzip to the folder that you want
  2. Or you can clone the repository through the command line:
$ git clone https://github.com/premium-minds/billy.git

Install Billy-Core

To install billy-core go to billy's folder and execute the following steps:

  1. Enter in the billy-core folder:
    $ cd billy-core
  2. Compile, execute the tests and install billy-core in a local repository.
    • 2.1 Build the project: 
      $ mvn compile
    • 2.2 Compile and run the project tests: 
      $ mvn test
    • 2.3 Build the project jar: 
      $ mvn package
    • 2.4 Install the project jar in a local repository: 
      $ mvn install
