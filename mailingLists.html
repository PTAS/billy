Mailing Lists

For issues with the Billy aplication you should submit your bug report to the Billy Issue Tracker

Rules to create an issue

  • 1. You should check if the bug has already been reported, to avoid duplicated issues, and check if the problem has already been solved.
  • 2. The issue must be labeled.
  • 3. Share as much information as possible, like your operating system and version, the version of Billy, and other information that you think may be important.
